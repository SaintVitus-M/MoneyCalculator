Welcome to Currency Converter!

This application allows you to easily convert currencies in real time using the Frankfurter API.

Stay up-to-date with the latest exchange rates and effortlessly convert between a wide range of currencies.

Features:

  • Access accurate, real-time exchange rates.
  • Perform quick and simple conversions between currencies.
  • Explore historical rates for better insights.

Start by selecting the currency you want to convert from and the target currency.
